Solution: Untar it.

Its easier to use a script to open them once you know the file naming convention is 1000.tar 999.tar 998.tar etc with filer.txt and the next number.tar.

```
import tarfile

for x in range(1000,0,-1):
        #print(x)
        filename = str(x)+".tar"
        #print(filename)
        tar = tarfile.open(filename)
        tar.extractall()
        tar.close
```
